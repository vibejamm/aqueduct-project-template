## Quick Start

```bash
# 1. Clone
git clone <repo-url>
cd your-project

# 2. Install
npm install
cd functions && npm install && cd ..

# 3. Firebase config
firebase init

# 4. Local dev
firebase emulators:start  # Terminal 1
npm run dev              # Terminal 2

# 5. Deploy
git push origin main
```
